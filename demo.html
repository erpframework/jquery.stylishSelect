Demo: jquery.stylishSelect

You can use CSS to set the style of the text label, the open button, the drop-down, and the options inside the drop-down, in normal, hovered, and open states. This example shows:
  • Hiding the border, but not the arrow, in the normal state.
  • Showing the border (and arrow), when the select is hovered.
  • Showing the border (and arrow), when the select is open.
But any other styles may easily be applied to these states. The options in the select can also be styled in the normal, hovered, and selected states. You can also tinker with the style of the ul for placement, borders, etc of the drop-down.
